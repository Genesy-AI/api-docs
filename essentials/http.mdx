## Overview

The Genesy API is a RESTful API that can be accessed directly using HTTP requests from any programming language or tool like cURL.

## Quick Start

All requests require an API key in the `x-api-key` header:

```bash
curl https://openapi.genesy.ai/endpoint \
  -H "x-api-key: gsk_your_api_key_here" \
  -H "Content-Type: application/json"
```

## Language Examples

<CodeGroup>

```python Python
import requests

headers = {
    "x-api-key": "gsk_your_api_key_here",
    "Content-Type": "application/json"
}

response = requests.get("https://openapi.genesy.ai/endpoint", headers=headers)
data = response.json()
```

```javascript JavaScript
const response = await fetch('https://openapi.genesy.ai/endpoint', {
  headers: {
    'x-api-key': 'gsk_your_api_key_here',
    'Content-Type': 'application/json'
  }
});

const data = await response.json();
```

```php PHP
<?php
$ch = curl_init();
curl_setopt($ch, CURLOPT_URL, "https://api.genesy.ai/endpoint");
curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
curl_setopt($ch, CURLOPT_HTTPHEADER, [
    "x-api-key: gsk_your_api_key_here",
    "Content-Type: application/json"
]);

$response = curl_exec($ch);
$data = json_decode($response, true);
curl_close($ch);
?>
```

```ruby Ruby
require 'net/http'
require 'json'

uri = URI('https://openapi.genesy.ai/endpoint')
http = Net::HTTP.new(uri.host, uri.port)
http.use_ssl = true

request = Net::HTTP::Get.new(uri)
request['x-api-key'] = 'gsk_your_api_key_here'
request['Content-Type'] = 'application/json'

response = http.request(request)
data = JSON.parse(response.body)
```
